YO SOY — Glitches & Faults Log

STATUS: RESOLVED

Glitch 01 — Database State Ignored

The instrument formerly ignored questionnaire values on startup. Hydration logic now forces the oscillator and UI to match the stored symbolic state.

STATUS: RESOLVED

Glitch 02 — Persistent Guest Status

Permission denied errors in the Firestore rules prevented roster lookups. Rules have been updated to allow document ID get requests for the roster.

STATUS: ACTIVE (STAGE 2)

Glitch 03 — Propagation Delay

In high-traffic environments, the onSnapshot broadcast may experience 1-2 second latency. This creates a staggered "wave" effect when Force Sync is toggled, which has been preserved as a pedagogical artifact of system friction.

STATUS: ACTIVE (STAGE 2)

Glitch 04 — CSS Collapse

During the integration of the ADSR controls, the custom range-input styling was susceptible to browser-default overrides. Re-injection of -webkit-appearance: none was required for Image A parity.
