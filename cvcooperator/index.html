Product description: CVCooperator is a cooperative voltage instrument for Eurorack that transforms rigid control signals into expressive, time-shaped motion. Rather than generating modulation, it negotiates between incoming voltage, human gesture, and temporal behavior using passive analog circuits and light-controlled resistance. The result is a living control surface where logic becomes movement.

Passive / Gate-driven Eurorack utility module
16 HP · 8 Inputs · 8 Outputs · No rack power required

This module converts rigid gates and CVs into organic, slow, rubbery control voltages using purely analog means: resistors, LDRs (vactrol-style), capacitors, and human-controlled attenuation.

1. Concept

Digital logic is fast and sharp. Music often is not. This module lives between those two worlds.

  • Input: gates, triggers, stepped CV (0–5 V)
  • Output: smoothed, lagged, human-scaled CV
  • Control: passive analog behavior + optional MCU-driven LEDs

2. System Architecture

  • No ±12 V Eurorack power used
  • ESP32 / Arduino powered over USB (optional)
  • All rack interaction is voltage-referenced only
  • Safe to experiment, safe to hot-patch

3. Single-Channel Core Schematic

This cell is replicated 8 times.

IN (0–5V) | [1k–4.7k] | +---- Modulation Node -----> Attenuation Pot -----> OUT | | | [LDR] | | | GND | +---- [1µF – 4.7µF] -----> GND (optional smoothing capacitor)

Component Notes

  • Series resistor: protects inputs and limits current
  • LDR: acts as a slow, nonlinear variable resistor
  • Capacitor: creates lag / glide (RC low-pass)
  • Pot: manual attenuation and performance control

4. What the Capacitor Does

The capacitor integrates voltage changes over time. Instead of jumping instantly, the output moves gradually.

  • Small value (1µF): subtle smoothing
  • Medium (2.2–4.7µF): envelope-like motion
  • Large values: slow, drifting modulation

Combined with an LDR, the result is non-linear and musical. The rise and fall times are different — like acoustic motion.

5. Using Vactrols / LDRs

An LED shining on an LDR forms a voltage-controlled resistor.

  • LED brightness controlled by gate, PWM, or MCU
  • LDR resistance changes slowly and imperfectly
  • Each channel is slightly unique (this is a feature)

6. MCU Integration (Optional)

An ESP32 or Arduino can drive LEDs only. It never touches Eurorack voltage directly.

  • PWM → LED → LDR
  • USB power only
  • Serial / MIDI / OSC from Max or Pure Data

7. Front Panel Layout (Suggested)

  • Top row: 8 inputs
  • Middle: 8 attenuation pots
  • Optional switches: cap in/out or mode select
  • Bottom row: 8 outputs

8. Cautions & Good Practice

  • Do not feed ±12 V signals into this module
  • Always include the series resistor
  • Hot glue Dupont cables only for prototyping
  • Expect channel variance — calibrate by ear

9. Philosophy

This is not a precision tool. It is a translator between logic and gesture.

Build one channel. Patch it. Listen. Then replicate.
